# RRT Kamera

Tactical camera HUD with target tracking overlay. Inspired by Panopticore Target Viewer.

## Features
- Live camera feed with cyberpunk HUD
- Fake target tracking system
- Telemetry display
- Thermal mode toggle
- PWA ready - installable as APK

## Setup
1. Clone repo
2. Host via HTTPS (required for camera)
3. Open on mobile device
4. Grant camera permission
